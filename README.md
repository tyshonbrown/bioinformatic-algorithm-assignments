# CMSC423 Programming Assignments

This repository contains programming assignments from my CMSC423: Bioinformatic Algorithms, Databases, and Tools course.

The assignments focus on implementing common bioinformatics and algorithmic techniques, including:

* Suffix trees
* Motif finding
* Minimum skew
* Lloyd’s algorithm
* KMP failure/sp values
* Hierarchical clustering
* Pattern finding
* DNA letter counting
* Burrows-Wheeler Transform
* BLOSUM62 alignment
* Local sequence alignment
* Cyclopeptide sequencing
